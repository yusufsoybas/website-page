Yeditepe University

1) Choose a different topic (it can be any topic, but not Yeditepe University). For example, a car brand, a football team, a fashion house, food recipes, history, etc.

2) Use 7 slider images (inside div elements), with 7 dots (inside span elements) below the image slider. If you want, you may use the same 7 images that you chose for Exercises 1 and 2, but it will be better if these images will be about the topic that you chose for your web pages (e.g., Ferrari images will be better in the slider if the web pages are about Ferrari cars). The slider images must have a width of 1100 pixels and a height of 400 pixels. You must accordingly modify the corresponding internal CSS value, to allow a width of 1100 pixels in the image slider.

3) You are allowed to copy-paste the text from existing websites. Fully change the text in the sample paragraphs with text about the subject that you chose for your web pages.

4) You must create three new web pages (in this example, you must create news.html, contact.html and about.html). Keep the names "Home" (default.html) and "Contact" (contact.html) unchanged in the topnav, while you may change "News" (news.html) and "About" (about.html) with different names if you want. You may copy and paste the HTML code of the home page (default.html) when creating the new pages. Don't forget to modify the value in the "title" tag underneath the "head" tag (which will appear inside the browser tab), and don't forget to move the class "active" next to the correct link in the "div" element with the class "topnav".

5) Change the font colors, background colors, font family, font size and other CSS values according to your own preferences.

6) Add at least 2 "images" and 1 "YouTube video" to each of the four pages. You must save the images inside the folder of the exercise before adding them to the web pages.

7) Add at least one "table", one "form" and one "Google map" to the "Contact" page. Change the Google map, which has to be about the topic that you chose.

8) Add your name and surname next to the Copyright symbol inside the "footer" element, at the bottom of the page, in each of the four pages.

9) The deadline to submit your homeworks is Sunday, January 16, 2022, at 23:59 Turkey time. You must compress the homework folders in RAR or ZIP format and send them to me via e-mail, to the address mustafa.kazancigil@yeditepe.edu.tr
