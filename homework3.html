In this exercise, you will need two jQuery lines for the two img (image) elements, and two jQuery lines for the two a (hyperlink) elements. First select all img elements with the class name cars, then use the jQuery attr() method chained with the eq() method to add index numbers for changing the src attribute value of "lamborghini.jpg" with "ferrari.jpg" (in the first image), and the src attribute value of "bugatti.jpg" with "koenigsegg.jpg" (in the second image). Also change the title attribute value "Lamborghini" with "Ferrari" (in the first image), and the title attribute value "Bugatti" with "Koenigsegg" (in the second image). Change the width attribute value "300" with "360" (in both images), and the height attribute value "200" with "240" (in both images). After completing the values of the attr() method, chain it with the css() method to change the border values as "10px solid red", and then chain it with the slideUp() and slideDown() methods, giving 1000 milliseconds for both slideUp() and slideDown() (in both images). Afterwards, select the two a elements (hyperlinks) with the class name carlinks and chain both of them with the eq() method to add the necessary index numbers for changing the href attribute value "http://www.lamborghini.com" with "https://www.ferrari.com" (in the first image's hyperlink); and "http://www.bugatti.com" with "http://www.koenigsegg.com" (in the second image's hyperlink).
